# 🚀 ParaBank Automation Framework

## 📌 Overview
Automation framework built using Selenium, Java, and TestNG with real-world design practices.

## 🛠 Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Apache POI (Excel)
- Extent Reports
- Allure Reports

## 📂 Features
- Page Object Model (POM)
- Data-driven testing using Excel
- TestNG suites (Smoke & Regression)
- ThreadLocal WebDriver (parallel-ready)
- TestNG Listeners (screenshot on failure)
- Retry Analyzer for flaky tests
- Extent & Allure Reports

## ▶️ How to Run
```bash
mvn test
```

## 📊 Reports 
- Extent Report: reports/index.html
- Allure Report: allure serve allure-results
